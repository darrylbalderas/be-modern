# be-modern

## Instructions

1. Create and start virtual environment

   - ```bash
     python3 -m venv be-modern
     source be-modern/bin/activate
     ```

2. Install dependencies

   - ```bash
     pip install -r requirements.txt
     ```

3. Start development server

   - ```bash
     make dev-start
     ``
     ```
